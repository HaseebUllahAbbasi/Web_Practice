Angular

  • Handle User input
  • Communicate with backend services

Node JS

Server Side lib : JS on server

  • Listen ot requests and send requests
  • Execute server side logic
  • interact with databases and files
  • Alternative of the php , Ruby and Rails and java

Express

Node Framework : simplifies writing server side code ,Logic

  • Middleware based:Funnel Requests through functions

Mongo DB

A noSQL Database , stores Documents in collection instead of records

    • easily connected to node/express not to angular

A power full Database which can be used in node js
